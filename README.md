# DIA - Digital Image Analyzer

### It's a powerful Java-based app that makes it easy to manage and optimize your image collections.

One of the standout features of DIA is its ability to compress multiple images in a directory with a specified percentage reduction. This is incredibly useful if you're looking to optimize image sizes for web or mobile applications. But that's not all - DIA also includes a smart duplicate `image detection algorithm` that compares images in a directory and automatically deletes duplicates. 

!The best part? It provides an `in-depth report` detailing which images were deleted and why, so you can be sure that you're not accidentally deleting anything important.
